# EduLex

EduLex is an AI-driven multimedia learning assistant designed to support dyslexic learners
by transforming text-based prompts into accessible, multimodal educational content.

## Current Features
- Text input interface
- Semantic keyword extraction
- Dyslexia-friendly UI

## Tech Stack
- Python
- Streamlit
- NLTK
